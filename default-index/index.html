本题是一个典型的文件上传漏洞题目。

不选择文件直接点击“upload”按钮上传，页面显示报错。
Fatal error: Uncaught ValueError: Path cannot be empty in /var/www/html/upload.php:19 Stack trace: #0 /var/www/html/upload.php(19): exif_imagetype('') #1 {main} thrown in /var/www/html/upload.php on line 19

发现该网站使用 exif_imagetype() 函数来判断上传文件的类型，而该函数是通过读取一个图像的第一个字节并检查其签名来判断一个图像的类型的。

多轮测试发现该网站只能上传图像文件。

图片马

那我们同样也可以通过图片马的方式来绕过该网站限制。

图片马的制作有以下几种方案。

编辑器添加数据

准备一张图片，用记事本或者010editor等编辑器直接打开，在后面增加一句话木马，此时你得到了一个图片马，然后修改图片文件名的后缀为php上传即可。

burp抓包添加数据

在上传图片文件时，用bp抓取请求包，修改图片文件名后缀为 .php ，在图片文件内容后面增加一句话木马后放开请求即可。

Windows命令制作

copy x.png /b + 1.php /a shell.png
其中 x. png 是一个普通的 png 图片文件，1.php 是 webshell 文件。
该命令就是将以二进制方式读取的 x.png 文件内容和以 ASCII 编码方式读取的 1.php 文件内容合并，生成新的文件 shell. png 就是图片马了，最后修改图片文件名的后缀为php上传即可。

一句话木马文件上传后使用中国蚁剑连接目标服务器在根目录可以找到 flag 文件。

麻将

分类于 Misc

麻将，四人骨牌博戏，流行于华人文化圈中。起源于中国，粤港澳及闽南地区、福州地区俗称麻雀。麻将是由中国古人发明的博弈游戏，娱乐用具，一般用竹子或骨头或塑料制成小长方块，上面刻有花纹或字样。

常常忘记以下几个要点，故记录于此。

抓牌顺序

庄家掷出两个骰子，确定庄家拿牌的方向和起始位置。

方向

从掷骰子的人开始按逆时针方向数，自己是1、5、9，右手边的那位是2、6、10，对方是3、7、11，左手边的是4、8、12，简称“左四右二”。

起始位置

选择好正确的牌堆后，我们通常按掷大拿小的方式取牌，比如骰子显示是3与6，表明是在右手方（6比3大）门前拿牌，从对方的右手往左手方向数（也就是我们说的掷色子的人的顺时针方向数）3墩牌。

从庄家开始逆时针方向拿牌，每人每次摸2墩（4张牌），每人摸取3次后，庄家摸1张，跳过1墩再摸一张，其他玩家从牌墙上按顺序（指牌前玩家正面的右手往左手方向）摸一张，最后庄家累计14张手牌，闲家各13张手牌。

出牌顺序

从庄家开始，按照逆时针方向依次出牌。

拿码

广东麻将获胜后，赢家会有马的奖励，所谓马就是你自摸的那张牌后面的几张，如果约定好几个马自摸的人就从牌墙上按顺序多摸几张牌作为马进行计算。

按照这把牌庄家（也就是上把的赢家）的位置逆时针排，四个玩家分别为东（庄家）、南、西、北。

按照多摸的牌（即为马）牌面上的数字计算中马的个数，赢家若在上一步骤中排为“南”，则“二六南发”作为中马数字，马中出现这些数字即为中马，具体中马数字如下表：

  • 东 一五九中东
  • 南 二六南发
  • 西 三七西白
  • 北 四八北

假设底码为一个筹码，如果中一个马，就是输家各付两个筹码，如果中两个，输家就各付三个筹码，以此类推。

注：不同地域规则有所不同，仅供参考。

本地方案

本教程为Windows11搭建本地基岩版服务器的方案。

Minecraft服务端 - Minecraft官网
Minecraft基岩版服务端BDS下载 - MeteorMC

下载Windows系统基岩版服务端

开放端口

在系统设置中找到“高级安全 Windows Defender 防火墙”
在 “入站规则”当中新建规则如下：
名字：自定义
协议类型：UDP
本地端口：特定端口19132,19133
远程端口：所有端口

运行服务器

双击打开“bedrock_server.exe”即可运行服务器
关闭窗口即可关闭服务器

配置服务器

总体配置

server.properties 总配置文件

权限配置

permissions.json 用户权限配置文件

白名单

whitelist.json / allowlist.json 白名单配置文件

存档

worlds文件夹

连接服务器

保证服务端与客户端版本一致才可以成功连接。
查看开启服务端设备的ip信息，记录下ipv4地址。
“Win+R”运行-“cmd”命令提示符-“ipconfig/all”查看ip信息

Android/iOS客户端

打开客户端连接上述记录的ip地址即可

Windows客户端

打开注册表编辑器

按照路径依次展开文件夹
计算机\HKEY_CURRENT_USER\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\CurrentVersion\AppContainer\Mappings\
Ctrl+F搜索”Minecraft for Windows 10”并选中

复制路径中S开头的一串字符（位于路径的最后一级）

将上述复制的字符接在下面的命令中
CheckNetIsolation.exe loopbackexempt -a -p=

以管理员身份运行命令提示符

输入上述命令并回车确定

打开客户端访问127.0.0.1或者上述记录的ip地址均可
